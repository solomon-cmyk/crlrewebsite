# Gallery images

Save **at least 3 images** here. More is fine.

## Good subjects
- Residence interior
- Exterior / villa
- Pool, amenities, dining, spa, or ocean view

## Requirements
- Minimum 1600px on longest side
- JPG or WebP
- Pull from existing Maravé website files / brand assets when possible

## Naming
`marave-gallery-01-[description].jpg`
`marave-gallery-02-[description].jpg`
`marave-gallery-03-[description].jpg`

## Also add to `../../image-manifest.csv`
Caption + alt text for each file.
